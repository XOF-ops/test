## Usage

### Initialize System
```bash
python master_brain.py
```

### Scan Input for Patterns
```bash
python master_brain.py --scan "I feel alone and trapped in survival mode"
```

### Simulate Friction (Rate Limiting)
```bash
python master_brain.py --scan "blocked by limits" --rate-limited
```

### Propose Governance Amendment
```bash
python master_brain.py --amend "Strengthen community bonds through weekly checkpoints"
```

## Pattern Recognition

The engine detects structural tension through trigger words:

- **Void**: but, however, impossible, conflict
- **Sacrifice**: cost, sacrifice, price, pain
- **Fear**: afraid, fear, alone, trapped, anxiety
- **Glitch**: distraction, apathy, survival, inflation
- **Friction**: limit, quota, blocked, error, 429
